   - 新增 CSS：`.lightbox-overlay`、`.lightbox-close`（全屏遮罩 + 毛玻璃背景）

- 点击遮罩或关闭按钮 → 遮罩淡出，恢复页面滚动
- 按 ESC 键 → 快速关闭
- 背景魔法阵（pointer-events-none）不受影响
